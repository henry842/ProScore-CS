## Executive Summary | Resumo Executivo

**EN**

Credit scoring, classification logic and customer risk indicators. This project was organized as a portfolio-ready case study: it explains the objective, the analytical path, the visual evidence and the practical interpretation behind the result.

Main objective: Structure a credit scoring project that explains inputs, risk signals and expected model behavior.

**PT-BR**

Score de credito, logica de classificacao e indicadores de risco do cliente. Este projeto foi organizado como um estudo de caso pronto para portfolio: explica o objetivo, o caminho analitico, as evidencias visuais e a interpretacao pratica por tras do resultado.

Objetivo principal: Estruturar um projeto de score de credito que explique entradas, sinais de risco e comportamento esperado do modelo.
